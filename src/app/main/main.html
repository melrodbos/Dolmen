Easy To Use Features

Add / Remove Properties

dolmen allows you to add and create detailed profiles for one or multiple properties. You can also conveniently edit and remove a property in one click.

Add / Remove Tenants

Add one or multiple tenants per property. Edit tenant's name and contact information. Remove tenants from a listing when the lease is up.

Manage Requests

Keep maintenance requests organized in different panels. Distribute requests by date created. Active requests panel will show a chronological list of editable requests. Add your seal of approval by checking off a completed request.

Request Reports

Generate reports based on maintenance requests history.
